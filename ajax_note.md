## AJAX （Asynchronous Javascript and XML ）
### Preview 
- Ajax 的作用： 使用JS 代码，获取服务器的数据
- Ajax 的效果： 在不刷新整个页面的情况下，通过一个url 地址获取服务器的数据，然后进行页面的**局部刷新** **异步刷新**
- 实例：评论异步加载（单词只加载若干评论，“加载更多”按钮）/ 用户名的验证（服务器通过对比数据库判断用户名是否可用
