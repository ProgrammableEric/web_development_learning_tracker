版权信息

alt, title 的区别

图片中的 alt 属性是在图片不能正常显示时出现的文本提示。 图片中的 title 属性是在鼠标在移动到元素上的文本提示。

span

span 为内容添加行内容器，相对的，div 则添加块级容器。但是，他们都不赋予内容具体的语义。使用span 来组合行内元素，以便CSS格式化。

left, right, top, bottom

Position 为 Static，他们不起任何效果

Box-sizing

默认值为content-sizing, 即对元素本身尺寸限定。可改为border-boxing。

Padding

应用在一个元素上的 padding 会影响它在网页上的尺寸，而不会影响不同元素之间的距离。

Margin的重叠问题

在以下情况下，margin 会重叠：

  • 相邻的兄弟姐妹
  • 完全空盒子
  • 父元素和第一个或最后一个子元素

float 与 position 的关系

经常用于父元素通过浮动定位，而其子元素需要绝对定位，这个时候就需要将父元素设置为相对定位，这时相对定位并不会改变浮动属性。 div｛float :left;position:relative｝这样是可以的 div｛float :left;position:absolute｝不允许，绝对定位会使浮动失效

last child

父元素的最后一个子元素，用子元素标签选择 e.g. /* Selects any p that is the first element among its siblings */ p:first-child { color: lime; }

伪类伪元素

书写时引号和伪类中间不能加空格 ！！！
